# Twig Extension

By this project, my most-used Twig extensions are bundled as a composer package.

## Usage

```php
// ...
$twig = new Twig_Environment( $loader, $options );
$twig->addExtension( new use macwinnie\TwigExtensions\All() );
// ...
```
